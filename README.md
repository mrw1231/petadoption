# Furry Friend Finder

Furry Friend Finder is here to help you find your new best four legged friend! You are able to see available pets. If one peaks your interest, you can sign up and reserve to meet the pet.

## Technologies Used

- MongooseDB
- Express
- React
- Node.js
- CSS
